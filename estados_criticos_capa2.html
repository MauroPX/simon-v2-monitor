WCAG 2.2 AAA — diferencias clave vs AA
Contraste texto: 7:1 mínimo
AAA exige 7:1 para texto normal (AA era 4.5:1). Verificar todos los tokens de Simon #00ffc2 sobre #080808.
Focus appearance (2.4.13)
El área del focus ring debe tener ≥ perímetro del componente × 2px. No solo un outline visible — debe tener área calculada.
Dragging alternative (2.5.7)
Drag & drop de paneles DEBE tener botones ↑ ↓ como alternativa accesible. Obligatorio para el layout reordenable.
Target size: 44×44px mínimo
AAA amplía el target mínimo. Todos los botones del dashboard, incluyendo controles del mapa, deben cumplir 44dp.

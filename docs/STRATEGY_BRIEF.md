## Propuesta de valor diferencial

**Simon Monitor** es el único Control Room Component de flota que:
1. Prioriza la experiencia del operador en turnos 24/7 (dark mode, fatiga visual)
2. Nunca muestra una pantalla en blanco ante errores de conectividad
3. Es 100% operable por teclado (WCAG 2.1 AA) — competidores no lo hacen
4. Tiene un roadmap transparente de 4 capas documentado en la propia app

## Oportunidades vs competidores

| Debilidad competidor | Solución Simon Monitor |
|---|---|
| Geotab: curva de aprendizaje alta | UI minimalista, solo 4 métricas clave |
| Samsara: sacrifica densidad de info | dl/dt/dd con jerarquía visual clara |
| GPS Trackit: baja accesibilidad | WCAG 2.1 AA verificado con jest-axe |
| Todos: error state = pantalla blanca | Última posición conocida siempre visible |

## Hipótesis de PMF
"Los operadores de flota en turnos nocturnos prefieren una herramienta
con dark mode nativo, actualización automática y estados de error empáticos
sobre dashboards con más features pero peor experiencia en condiciones de fatiga."
